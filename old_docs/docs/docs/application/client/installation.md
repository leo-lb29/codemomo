## Installation 

:::danger
Vous devez avoir préparer votre rapsberry pi voir : Raspberry PI
:::

Le code ce trouve dans dans l'archive du repo gitlab `/src/app/host`

ce dossier contient un .tar que vous allez venir placer à la racine `/home/client/` d'un raspberry pi servant de client

avec `scp <source> <user>@<ip|hostname>:~`

une fois sur le rapsberry pi taper 

`tarball -X client.tar`
