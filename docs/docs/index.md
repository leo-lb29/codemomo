# Introduction
Dans le cadre de l'Unité d'enseignement PRJ1401 de L2 SNIO, il a été demandé de réalisé un travail autour du traitement audio, le projet c'est divisé sur 3 ans avec pour objectif le traitement d'un signal audio, on a eu donc la V1 qui ce concentre sur le signal audio et la v2 actuel ce concentre sur la diffusion de ce signal sur un réseau IP.
Cette docuementation est divisé en plusisuers partie:
La présentation du matériel physique utilisé pour l'étude et le cas d'utilsiation
L'étude de la diffusion d'un signal sur un réseau IP
Le cas d'utilisation
