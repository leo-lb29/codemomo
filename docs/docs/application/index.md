# Introduction

Pour le cas d'utilisation nous somems parties sur un système de conférence :
l'application : une seule personne parle à la fois

Un maître de conférence : Gère qui parle, route l'audio sur le réseau, peut parler, peut entendre

Client : Peut demander la parole pour parler, peut entendre
